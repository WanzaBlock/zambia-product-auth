## Frontend (Vanilla HTML / CSS / JavaScript)

Web interface for manufacturers and consumers.

Features:
- Consumer product verification via QR code scan or manual item ID entry
- Real-time blockchain verification result with genuine / clone / expired status
- Blockchain proof card showing transaction hash and Basescan link
- Manufacturer login and multi-tenant dashboard
- Product batch registration form
- Counterfeit alert feed with resolve actions
- Recent scan activity feed
