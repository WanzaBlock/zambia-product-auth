# WanzaBlock Backend

## Backend API (Node.js / Express)

Node.js backend responsible for:
- Handling manufacturer product batch registration
- Generating product QR codes
- Logging scan requests with GPS coordinates
- Interacting with the blockchain smart contract
- Real-time counterfeit geo-detection

## Example Endpoints
```
POST /mfr/register-batch
POST /verify
GET  /mfr/alerts
```
